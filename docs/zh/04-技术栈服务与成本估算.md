# 技术栈、服务与成本估算

## 1. 说明

这份文档用于估算 SBBC 可能需要的第三方服务、技术栈组合和基础成本。

原则：

- 前期保持轻量
- 不要一次性购买所有服务
- 与版本规划同步采购

## 2. 推荐基础技术栈

### 核心层

- Vercel：应用部署
- Cloudflare：DNS、WAF、域名和网络层
- Supabase：数据库、认证、存储、Realtime
- Resend：事务邮件
- Stripe：在线支付

### 在线课程阶段再引入

- 视频托管：Cloudflare Stream 或 Mux
- 直播基础设施：LiveKit 或先接第三方会议工具

## 3. 分阶段服务建议

### V1

- Vercel Pro
- Cloudflare Pro 或 Free + 必要增值项
- Supabase Pro
- Resend 免费或低阶付费
- Stripe 按交易收费

### V2-V3

- Vercel Pro
- Cloudflare Pro
- Supabase Pro，并视增长提升 compute 或 storage
- Resend 付费
- Stripe

### V4

- 增加视频托管
- 增加直播服务
- 提高存储与带宽预算

## 4. 关键服务价格快照

以下价格应在正式采购前再次确认。

- Pro：`20 美元 / 月 / developer seat`
- 官方来源：[Vercel pricing](https://vercel.com/pricing)

### Supabase

- Pro：`25 美元 / 月起`

- Custom domain add-on：`10 美元 / 域名 / 月 / 项目`
- 官方来源：[Supabase pricing](https://supabase.com/pricing)

- Scale：`90 美元 / 月，100,000 封邮件 / 月`

- Resend 还有更低层级的方案，但这次抓取到的官方页面中明确暴露的是 Scale 档位

## 5. 月度成本场景估算

### 场景 A：V1 轻量上线

假设：

- 一个正式生产应用
- 流量较低到中等
- 以官网、表单、内容、基础 CRM 为主
- 暂不做平台内视频和直播

月成本预估：

- Vercel Pro：`20 美元`
- Supabase Pro：`25 美元`
- Cloudflare Pro：`20 到 25 美元`
- Resend：`0 到 30+ 美元`
- Stripe：按交易收取手续费

合计：

- 大约 `65 到 100+ 美元 / 月`，不含支付手续费

### 场景 B：V2-V3 运营型平台

假设：

- 管理后台、教师端、学生 Portal 均开始使用
- 邮件、存储和数据库访问增加

月成本预估：

- Vercel Pro：`20 到 40 美元`
- Supabase：`25 到 100+ 美元`
- Cloudflare Pro：`20 到 25 美元`
- Resend：`20 到 90+ 美元`
- Stripe：按交易收费

合计：

- 大约 `85 到 255+ 美元 / 月`，不含支付手续费

### 场景 C：V4 含在线课程

假设：

- 有录播课程
- 有直播课
- 视频、存储和带宽需求显著增长

月成本预估：

- Vercel Pro：`20 到 40 美元`
- Supabase：`60 到 210+ 美元`
- Cloudflare：`20 到 25 美元`，外加可选媒体服务
- 邮件：`20 到 90+ 美元`
- 视频 / 直播：`50 到 500+ 美元`
- Stripe：按交易收费

合计：

- 大约 `170 到 865+ 美元 / 月`，不含支付手续费和高使用量超额部分

## 6. 推荐采购顺序

建议按以下顺序购买：

1. 域名与 DNS
2. 托管服务
3. 数据库与认证
4. 事务邮件
5. 支付网关
6. 视频与直播服务在 V4 再引入

## 7. 当前阶段通常不需要立即购买的服务

- 企业级可观测性套件
- 独立 BI 数据仓库
- 重型搜索服务
- 完整 LMS 商业套件
- 独立 ERP / 财务系统
