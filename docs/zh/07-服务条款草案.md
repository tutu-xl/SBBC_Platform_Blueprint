# SBBC 服务条款草案

> 本文件仅作为项目规划草案，不构成法律意见。正式上线前，建议由澳洲律师或合规顾问根据 SBBC 的实际主体、业务流程和付款规则进行审核。

## 1. 文件目的

本服务条款用于说明用户使用 SBBC 官网、学生端、家长 / 监护人端、教师端、报名流程、付款流程、直播课程、录播内容和相关服务时需要遵守的基本规则。

正式发布版本需要根据 SBBC 的真实公司主体、ABN / ACN、联系方式、退款政策、支付通道、课程交付方式和澳洲法律要求进一步完善。

## 2. 适用对象

本条款适用于：

- 浏览 SBBC 官网的访客
- 注册账号或报名课程的学生
- 为未成年学生注册、确认或付款的家长 / 监护人
- 使用平台工具的教师、助教和运营人员
- 访问 SBBC 内容、课程、活动、表单或线上服务的其他用户

## 3. 账号和信息真实性

用户在注册、提交咨询、预约试听、报名课程或付款时，应提供真实、准确、及时更新的信息。

用户应妥善保管自己的登录信息，不应将账号借给他人使用。

SBBC 可以通过用户提供的邮箱、电话、微信或其他联系方式，就咨询、报名、课程、付款、续费、缺勤、补课等事项联系用户。

## 4. 成年学生与未成年学生

SBBC 需要区分成年学生和未成年学生。

成年学生可以：

- 自行注册账号
- 自行报名课程
- 自行完成付款
- 自行管理订单、发票和收据

未成年学生可以：

- 在合适情况下创建或使用学生学习账号
- 发起报名 / 选择课程
- 查看课程、课表、课时、反馈、学习资料和积分

未成年学生不能独立完成课程付款。付款、服务条款确认、隐私政策确认和未成年人相关确认事项，应由家长或监护人完成。

## 5. 家长 / 监护人责任

对于未成年学生，家长或监护人负责：

- 确认学生报名课程
- 完成学生课程付款
- 同意服务条款和隐私政策
- 完成未成年人相关确认事项
- 接收付款、续费、缺勤、补课等通知
- 处理退款、发票、收据和账单问题
- 监督学生学习情况

## 6. 课程报名与付款

SBBC 应尽量在付款前清楚展示课程内容、价格、课时、授课方式和可用情况。

付款可能通过 Stripe 或 SBBC 后续选择的其他第三方支付服务完成。用户使用第三方支付服务时，也可能受到该服务提供商的条款和隐私政策约束。

通常情况下，SBBC 收到付款或确认订单后，报名才视为完成。

SBBC 应保存订单、付款、发票、收据、退款和调整记录。

## 7. 取消、补课与退款

SBBC 应单独发布退款与取消政策，或在正式服务条款中写清楚具体规则。

退款与取消政策建议覆盖：

- 试听课取消
- 正式课程取消
- 缺勤和未出席规则
- 补课资格
- 课程套餐有效期
- 退款申请流程
- 退款方式和处理时间

SBBC 的条款不应排除澳洲消费者权益法中不能被排除的消费者权利，包括适用于服务的消费者保障。

## 8. 直播课程与第三方工具

SBBC 可以使用 Zoom、腾讯会议、Google 服务、支付服务、邮件服务、托管服务、数据分析工具等第三方工具。

用户访问第三方服务时，可能也需要遵守该第三方服务自己的条款和隐私政策。

SBBC 应合理提供课程链接、访问说明和支持信息；用户也需要自行准备合适的设备、网络连接和登录信息。

## 9. 录播内容与学习资料

SBBC 可能提供录播课程、讲义、作业、课后反馈、证书和其他学习资料。

除非另有说明，这些资料仅供已报名学生个人学习使用。

未经 SBBC 许可，用户不得复制、下载、录屏、分享、出售、转载或分发 SBBC 的学习资料。

## 10. 用户行为要求

用户不得：

- 滥用网站或平台
- 尝试访问他人账号
- 上传违法、有害、误导或冒犯性内容
- 干扰课程、教师、学生或平台运营
- 未经许可复制或传播 SBBC 内容
- 利用 SBBC 服务进行垃圾信息、欺诈、骚扰或违法活动

如果用户违反条款，或造成安全、法律、付款、运营风险，SBBC 可以暂停或限制其访问权限。

## 11. SBBC Store

SBBC 主站可以展示周边商品，并跳转到独立的 `SBBC Store`。

除非后续另行扩展，SBBC 主平台不直接处理周边商品的下单、库存、发货和售后。

用户在独立 `SBBC Store` 网站完成购买时，应适用该商店自己的服务条款、隐私政策、付款规则和退款政策。

## 12. 知识产权

SBBC 拥有或合法使用官网内容、课程资料、品牌资产、视频、设计、文字、图片和平台内容。

用户提交的个人信息和学习相关信息仍按适用法律处理；但用户授权 SBBC 在提供服务、管理报名、交付课程、处理付款和运营平台所需范围内使用这些信息。

## 13. 隐私

SBBC 如何处理个人信息，应在隐私政策中说明。

用户在注册、提交咨询、报名课程或付款前，应阅读隐私政策。

## 14. 服务或条款变更

SBBC 可能不时更新课程内容、平台功能、价格、政策或本服务条款。

如果变更会对用户产生重大影响，SBBC 应在可行情况下提前通知。

最新版本应发布在 SBBC 网站上。

## 15. 责任限制

SBBC 应以合理的谨慎和技能提供服务。

在法律允许范围内，SBBC 不对用户自身错误、用户提供信息不准确、第三方服务中断、设备问题、网络连接问题或超出 SBBC 合理控制范围的事件造成的损失承担责任。

本条款不应限制澳洲法律下不能被限制的用户权利。

## 16. 联系方式

正式发布版本应补充：

- SBBC 法律主体名称
- ABN 或 ACN，如适用
- 公司地址或邮寄地址
- 联系邮箱
- 电话
- 支持或投诉处理方式
